# AutoPilot IT – Free Password Vault

A free, local password vault for Windows that stores credentials securely on the user’s computer.

Designed for small offices that want a simple way to store shared or admin credentials without cloud sync or third-party services.

- Runs locally
- No cloud storage
- No data uploads
